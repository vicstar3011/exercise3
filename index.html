This is the world’s simplest website! This is the first exercise for our University of Waterloo web design class. Tuth be told this humble site is not that impressive but we have to take baby steps when learning HTML. Things start out fairly simple but as we learn different tags and their properties we’ll be able to structure content more efficiently. HTML vs. CSS In the first part of the semester we’ll focus entirely on structuring information with HTML. Only once we learn how to do that properly—it takes some practice—will we begin to apply graphic design principles to it. Learning proper HTML syntax is essential and your main job over the next month.
